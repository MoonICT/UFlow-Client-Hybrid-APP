#### Installation
``` bash
# install node moduel
$ npm install [module-name]
$ pod install

# RN link
# RN >= 0.60 자동링크(링크 필요없음)
# RN <= 0.59 수동링크(수동 링크 연결 시)
$ react-native link [module-name]
```
